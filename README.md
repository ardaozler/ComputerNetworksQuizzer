# Computer Networks Quizzer

Single-file **Computer Networks quiz app** for PoliTO exam practice.

## Usage
- Download the `.html` file  
- Open it locally in any browser  
- No server, no install, no dependencies

## Mobile
- Tested on **iPhone (iOS Safari)**  
- Need to download a free **HTML Viewer** app.
- Runs smoothly and offline

## For
Built for **Politecnico di Torino students** and peers.
